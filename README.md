** PROJECT TITLE: "EARLY STAGE DETECTION OF AUTISM SPECTRUM DISORDER USING DEEP LEARNING".

** DESCRIPTION: Autism Spectrum Disorder (ASD) is a neurodevelopmental condition characterized by challenges in social interaction, communication, and repetitive behaviors.This project is a deep learning-based system designed to detect Autism Spectrum Disorder (ASD) at an early stage using facial images and behavioral data. It uses a Convolutional Neural Network (CNN) model to automatically extract features and classify individuals as Autistic or Non-Autistic.
The system helps in faster and more accurate diagnosis, supporting healthcare professionals for early intervention.

** FEATURES: 
1.Upload image for ASD prediction
2.CNN-based image classification
3.Data preprocessing (resizing, normalization, augmentation)
4.Real-time prediction with confidence score
5.Displays result as Autistic / Non-Autistic
6.User-friendly web interface.

** TECHNOLOGIES USED:
-> Programming Language: Python
-> Framework: Flask
-> Deep Learning: TensorFlow, Keras
-> Libraries: NumPy, OpenCV
-> Frontend: HTML, CSS, JavaScript
-> Model: CNN (VGG16 Transfer Learning).

** HOW IT WORKS:
-> User uploads an image
-> Image is preprocessed (resize, normalize)
-> CNN model extracts features
-> Model predicts ASD or Non-ASD
-> Result displayed with confidence.

** ACHIEVED ACCURACY:85% – 92% (based on dataset & training)
Evaluated using:Accuracy, Precision, Recall, F1-Score, ROC Curve.

** OUTPUT:
Successfully classifies ASD vs Non-ASD
Displays:
✅ Green → Non-Autistic
❌ Red → Autistic
